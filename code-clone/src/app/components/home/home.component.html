Welcome to Code Clone Detector!

This is a tool that helps you determine if two pieces of code are clones of each other.

Get Started

Clone Types:

Type 1

Identical code fragments except for variations in whitespace (may also be variations in layout) and comments.

Type 2

Structurally/syntactically identical fragments except for variations in identifiers, literals, types, layout and comments.

Type 3

Copied fragments with further modifications. Statements can be changed, added or removed in addition to variations in identifiers, literals, types, layout and comments.

Type 4

Two or more code fragments that perform the same computation but implemented through different syntactic variants.
